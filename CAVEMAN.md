# 🪨 batch-tool, Cave Story Edition

## 🔥 Big Idea
Many caves. Many git repos. One chief no want walk cave by cave.

`batch-tool` = tribe drum + runner squad.
Chief shout one command. Many caves work at same time.

## ✅ Why Tribe Use It
- One shout, many repos move
- Consistent behavior across caves
- Faster batch work
- Less copy-paste mistake

## 🧭 Cave Picking Rules
Tool choose caves by selectors:
- `~label` include marked caves
- `!exclude` skip caves
- `+force` force include
- `~all` choose all caves

Chief target right caves fast.

## 🏃 Runner Behavior
Each runner take one cave:
- enter repo
- run task
- return result

Many runners move in parallel. Work finish faster, less human foot pain.
